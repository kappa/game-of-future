# Error And Recovery Log

Append one separate incident block per incident. Create the block when the
failure or pause first occurs. Never edit these instructions or the schema
below. Never write live incident values in this header. When the user responds,
or when an approved retry or resumption is recorded, update `User decision` and
`Resumption` inside that same incident block before continuing.
